# UI 필요 여부
기존 UI(결과 화면)의 레이아웃 버그 수정이므로 신규 UI는 필요 없음.

# 사용자 흐름
사용자가 라운드를 마치고 결과 화면을 볼 때, 쾌적하게 핏되는 화면을 경험.

# 화면 또는 명령 흐름
라운드 종료 -> `ResultView` 진입 -> 화면 렌더링 시 여백 및 스크롤바 없음.
