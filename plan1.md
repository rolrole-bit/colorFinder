# DYE MASTER 슬라이더 무한 대칭 루프 구현 계획 (plan1.md)

## 1. 작업 목적
현재 HSL 조작 슬라이더 중 S(채도)와 L(명도) 슬라이더는 위아래 물리적 한계(0% 및 100%)에 도달하면 드래그가 뚝 끊기며 멈추도록 설계되어 있습니다. 이를 무한 루프 형태로 개선하여 위아래 경계를 넘어가도 끊김 없이 자연스럽게 값이 왕복 순환(0 - 1 - 0 - 1 - 0)하며 이어지도록 만듭니다.

## 2. 핵심 기능
- **모든 슬라이더 무한 스크롤화**: H 슬라이더뿐만 아니라 S, L 슬라이더도 위아래 한계를 벗어나 드래그할 때 멈추지 않고 계속해서 무한 랩핑 순환되도록 합니다.
- **S, L의 대칭형 값 변환 (0 - 100 - 0)**:
  - S와 L은 양 끝단 색상의 이질감이 크기 때문에(예: 0% 회색 vs 100% 고채도), 랩핑 경계면에서 색이 뚝 끊기지 않도록 대칭(0 -> 100 -> 0) 왕복 형태로 값이 매끄럽게 반사 적용되도록 구현합니다.
- **끊김 없는 대칭 SVG 그라디언트 띠**:
  - SVG linearGradient 정의를 대칭 구조(0~50% 정방향, 50~100% 역방향)로 확장하여, 무한히 스크롤 하더라도 시각적으로 경계선이 튀지 않고 무한 반복되도록 통일합니다.

## 3. 입력과 출력
- **입력**: 플레이어 드래그 움직임 (`pointermove` 변위 `dy`), 셔플 애니메이션 구동 프레임.
- **출력**: 0~100 사이를 자연스럽게 왕복(Ping-pong)하는 아웃풋 값 및 끊김 없이 무한 스크롤되는 3단 슬라이더 그래픽.

## 4. 파일 구조
- [GameView.js](file:///e:/AI/DYE_MASTER/colorFinder/src/ui/GameView.js): S, L 슬라이더의 물리적 영역 2배 확장(0~200), `onChange` 대칭 변환 로직 탑재, SVG 그라디언트 및 테이프 구조를 3단 대칭형으로 통합 개편.

## 5. 핵심 모듈
- **Wrap Loop Engine**: `TapeSlider` 내부에서 드래그 변위값을 더해 0~1 범위로 랩핑하여 모든 슬라이더의 한계를 없앰.
- **Symmetry Calculator**: 물리값 `0~200`을 논리값 `0~100` 왕복 값으로 반사 변환하는 로직 (`val > 100 ? 200 - val : val`).

## 6. 실행 흐름
1. `TapeSlider` 객체는 들어오는 변위를 더하고 `Math.floor`를 빼 순수 백분율 루프를 돎.
2. S, L 슬라이더의 값 범위는 물리적으로 `0 ~ 200`으로 매핑되어 작동.
3. 슬라이더가 100을 넘어가 120이 되면 아웃풋은 80이 되어 자연스러운 대칭 감소 연출.
4. SVG Defs 내 `grad-s`와 `grad-l`이 대칭형 색상 키 프레임으로 렌더링되어 위아래 3단 테이프 결합 시 완전히 매끄러운 단일 무한 루프 형성.

## 7. 에러 처리
- 음수 오프셋 연산 시 모듈러 연산 안전장치 구현 (`(val + max) % max`).
- 인트로 믹싱(셔플) 시에도 대칭 루프 범위(`0~200`) 및 공식에 호환되도록 데이터 매핑 조율.

## 8. 테스트 전략
- H 슬라이더를 계속 아래로 내릴 때 무한 회전하는지 확인.
- S 슬라이더를 계속 내릴 때 채도가 빠졌다가(Gray) 다시 들어차는(Color) 왕복 연출이 튀는 그래픽 없이 무한히 반복되는지 검증.
- L 슬라이더 역시 화이트 -> 컬러 -> 블랙 -> 컬러 -> 화이트 형태로 자연스럽게 무한 왕복하는지 검증.

## 9. 완료 기준
- 모든 슬라이더에서 드래그 멈춤 한계점이 완전히 제거되어 무한 스크롤이 가능함.
- S와 L의 변색 경계가 뚝 끊기지 않고 0~100~0 대칭 왕복함.
